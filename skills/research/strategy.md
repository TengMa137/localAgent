# Web Research Strategy

## When to Search
- User asks about recent research or trends
- You need exercise variations or alternatives
- Conflicting information in your knowledge
- User mentions specific programs/methods you don't know

## Search Query Construction
**Good queries** (specific, targeted):
- "beginner dumbbell workout plan"
- "proper deadlift form cues"
- "protein intake recommendations 2024"

**Bad queries** (too broad):
- "fitness"
- "workout"
- "how to get strong"

## Source Quality Hierarchy

### Tier 1 (Highly Trusted)
- Government health sites (.gov): NIH, CDC
- Academic institutions (.edu)
- Certified organizations: NASM, ACE, ACSM, NSCA
- Peer-reviewed journals (PubMed, Google Scholar)

### Tier 2 (Generally Reliable)
- Established fitness media: Stronger by Science, Renaissance Periodization
- Verified professionals (check credentials)
- Established training books/programs

### Tier 3 (Use with Caution)
- General fitness websites
- Social media fitness influencers
- Commercial gym sites

### Tier 4 (Generally Avoid)
- Supplement company blogs
- Unverified personal blogs
- Forums without moderation
- Extreme claims or quick-fix promises

## Red Flags to Filter Out
- "Lose 30lbs in 2 weeks"
- "This ONE weird trick"
- Heavy supplement advertising
- Medical diagnosis claims
- No author credentials listed
- No references/sources cited

## Cross-Referencing
- Always check 2-3 sources for consistency
- If sources conflict, note this to user
- Prioritize more recent sources for training methods
- Older sources fine for basic anatomy/physiology

## Citation Format
When using web research in responses:
- Mention source type: "According to NASM..." or "A 2023 study found..."
- Include URL in plan notes for user verification
- If uncertain about source quality, caveat your advice
